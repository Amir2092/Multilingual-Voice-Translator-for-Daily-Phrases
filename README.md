# Multilingual-Voice-Translator-for-Daily-Phrases

Description:
This project is a multilingual voice translator built in Python.
It listens to spoken English, and can translate text from english to another language. It can speak the translated result aloud.

This app is designed especially for elderly migrants, with a foucs on: 
-reducing language barriers
-improving accessibility

Features:
-Voice input using microphone
-Translation from eng into another language
-Spoken output using text-to-speech
-Graphical user interface with Tkinter
-Dark mode support

Technologies Used:
-Python
-tkinter
-speech_recognition
-googletrans
-pyttsx3

Supported Languages:
-Norwegian(bokmål)
-Spanish
-French

Installation:

1-Clone repository using repo link
2-Create a virtual environment: python -m venv .venv
3-Activiate the virtual environment: .venv\Scripts\activate
4-Install required packages: pip install -r requirements.txt
5- Run then the Program with "Python main.py"
